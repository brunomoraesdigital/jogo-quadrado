JavaScript - DOM

SELEÇÃO BÁSICA DE ELEMENTOS

Método Descrição Exemplo
getElementById() Seleciona elemento pelo ID document.getElementById('header')
getElementsByClassName() Seleciona elementos pela classe document.getElementsByClassName('btn')
getElementsByTagName() Seleciona elementos pela tag document.getElementsByTagName('div')
getElementsByName() Seleciona elementos pelo atributo name document.getElementsByName('email')
querySelector() Seleciona o primeiro elemento que corresponde ao seletor CSS document.querySelector('.menu li')
querySelectorAll() Seleciona todos os elementos que correspondem ao seletor CSS document.querySelectorAll('div.box')

PROPRIEDADES DE DIMENSÃO E POSIÇÃO

Propriedade Descrição Exemplo
offsetWidth Largura total (conteúdo + padding + borda) elemento.offsetWidth
offsetHeight Altura total (conteúdo + padding + borda) elemento.offsetHeight
clientWidth Largura visível (conteúdo + padding) elemento.clientWidth
clientHeight Altura visível (conteúdo + padding) elemento.clientHeight
scrollWidth Largura total do conteúdo (incluindo área não visível) elemento.scrollWidth
scrollHeight Altura total do conteúdo (incluindo área não visível) elemento.scrollHeight
offsetTop Distância entre o elemento e o topo do offsetParent elemento.offsetTop
offsetLeft Distância entre o elemento e a esquerda do offsetParent elemento.offsetLeft
getBoundingClientRect() Retorna tamanho e posição relativa ao viewport elemento.getBoundingClientRect()

NAVEGAÇÃO ENTRE ELEMENTOS

Propriedade Descrição Exemplo
parentElement Elemento pai direto elemento.parentElement
children Elementos filhos (apenas elementos) elemento.children
firstElementChild Primeiro elemento filho elemento.firstElementChild
lastElementChild Último elemento filho elemento.lastElementChild
nextElementSibling Próximo elemento irmão elemento.nextElementSibling
previousElementSibling Elemento irmão anterior elemento.previousElementSibling

MANIPULAÇÃO DE CONTEÚDO

Método/Propriedade Descrição Exemplo
textContent Texto dentro do elemento (sem formatação) elemento.textContent = 'Texto'
innerHTML HTML dentro do elemento (com formatação) elemento.innerHTML = '<b>Texto</b>'
value Valor de inputs, selects e textareas input.value = 'Valor'
setAttribute() Define um atributo elemento.setAttribute('id', 'novoId')
getAttribute() Obtém o valor de um atributo elemento.getAttribute('class')
classList.add() Adiciona uma classe elemento.classList.add('ativo')
classList.remove() Remove uma classe elemento.classList.remove('inativo')
classList.toggle() Alterna uma classe elemento.classList.toggle('visivel')

CRIAÇÃO E MANIPULAÇÃO DE ELEMENTOS

Método Descrição Exemplo
createElement() Cria um novo elemento document.createElement('div')
appendChild() Adiciona um elemento filho pai.appendChild(filho)
removeChild() Remove um elemento filho pai.removeChild(filho)
replaceChild() Substitui um elemento filho pai.replaceChild(novo, antigo)
cloneNode() Clona um elemento elemento.cloneNode(true)

MANIPULAÇÃO DE ESTILOS

Método/Propriedade Descrição Exemplo
style Acessa/modifica estilos inline elemento.style.color = 'red'
getComputedStyle() Obtém estilos computados (incluindo CSS) getComputedStyle(elemento).fontSize
className Obtém/define a classe do elemento elemento.className = 'classe'

EVENTOS COMUNS

Evento Descrição Exemplo
click Clique do mouse elemento.addEventListener('click', funcao)
mouseover Mouse sobre o elemento elemento.onmouseover = funcao
mouseout Mouse sai do elemento elemento.onmouseout = funcao
keydown Tecla pressionada elemento.onkeydown = funcao
submit Formulário enviado form.onsubmit = funcao
change Valor alterado (input, select) input.onchange = funcao
